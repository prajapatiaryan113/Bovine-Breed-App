# Bovine-Breed-App
Bovine Breed Classifier – A Streamlit web app to classify cattle and buffaloes breeds using deep learning, save predictions, and manage records.
## Features

- User authentication (Sign Up / Login / Guest access)  
- Upload images or capture via camera  
- Predict bovine breed using a deep learning model  
- Edit prediction details: breed, height, weight, age, gender  
- Save predictions to a SQLite database  
- View all saved records with images  
- Update user profile (name, phone, address)  

## Technology Stack

- Python 3  
- Streamlit (frontend)  
- TensorFlow/Keras (deep learning model)  
- SQLite (database)  
- Pillow & NumPy (image processing)
